Projects

Final Bachelor Software Project

We worked for an assignment as a team of 11 developers and 2 artists on a project for the Geoscience department of Utrecht University. I spend a total of 430 hours on the project, and this is where most of my experience with React and typescript comes from. We made a Mobile Application, Web Application (Teacher Module) and setup a server. The Goal was to make a Mobile App for making geological fieldwork more fun and interactive by being able to place poi (points of interest) with images, and text which other people could see. A web app where the teacher could set up the environment and a server where all the data could be stored and requested by both parts. I worked almost exclusively on the Web application with 2 others. At the start of the project, I did follow an elaborate course on how to develop an android app. Which is where my limited experience with Kotlin comes from. For the project we worked with a large set of libraries since React is very barebones. For example, React-Router, React Query, React testing library, and Styled components. I learned a lot from this project and had a great time. I became better at approaching testing and improved my skills on working together on a large team.
